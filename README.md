#Rummy Game
A Rummy card game made for CSC 667/867

###For installing and running server:
```
git clone https://github.com/SFSU-CSC-667/term-project-team-5.git
cd term-project-team-5
npm install
npm run build
npm run start
```

###For playing game:
```
Go to http://localhost:3000 on browser
```
